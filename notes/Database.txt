Database

All dates are strings in YYYY-MM-DD

users
-----
1) ID
2) Display Name
3) Username
4) Password
5) User type (1 for normal user, 2 for admin)
6) Duty start date
7) Duty end date

exemptions
----------
1) ID
2) User
3) Start date
4) End date
5) Type

dutyPersonnel
--------------
1) id
2) userOnDuty
3) date
4) isManual (1 for true, 0 for false)
